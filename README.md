# Live Score

Live football scores built with Next.js, TypeScript, and Styled Components.

## Getting Started

```bash
pnpm install
pnpm dev
```

Open http://localhost:3000

## Tests

```bash
pnpm test
pnpm test:coverage
```

## Tech Stack

- **Next.js 16** — App Router, Server Components
- **Styled Components** — SSR via StyledComponentsRegistry
- **Barlow** — self-hosted via next/font
- **Vitest** — unit tests with React Testing Library
